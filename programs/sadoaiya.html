Sadoaiya

Sadoaiya is a .Net 6.0 program used for editing gmo models,
made mainly with the goal of making them work in Persona 4 Golden

Download

Features

Sadoaiya's easily configurable features include:

  • Applying all changes necessary to make a model work in P4G PC
  • Converting to a text format
  • Importing and extracting AMD files
  • Extracting and importing textures
  • Changing formats of arrays

Process of Importing a Model

WARNING: This semi guide does not cover the actual process of
getting your models ready, as it focuses only on the making them work in P4G part

Preregs:

  • .NET 6.0 (necessary in order to run sadoaiya)
  • Sadoaiya (preconfigured for P4G PC by default)
  • PSP converter tools (preconfigured for P4G PC)

Guide:

  • Export your model as FBX (preferably 2009 ascii)
  •         Make sure that your model is rotated 180 degrees on the Z axis!
  • Convert your model to gms using gmoconv.exe found in PSP converter tools with gmoconv.exe {yourFBX} -S
  • Apply desired changes to the gms file using a text editor
  • Convert your gms to gmo using gmoconv.exe with gmoconv.exe {yourFBX}
  • Fix your model for P4G by drag and dropping it on sadoaiya.exe or by using sadoaiya.exe {yourGMO} -A to save it as an amd
